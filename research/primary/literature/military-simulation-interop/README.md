# Military Simulation Interoperability Literature

This folder collects open technical and academic literature that helps explain
how military/distributed simulation standards are used in practice, especially
where official standard overviews alone do not show the operational design
patterns.

## Why These Papers Matter

These sources complement the standards pages by showing:

- how MSDL is used as a scenario-initiation and interchange format rather than
  as the whole interoperability stack
- how TENA, DIS, and HLA are bridged in operational live/virtual/constructive
  environments
- how engineering process, gateways, and common context requirements become
  unavoidable in real multi-system training deployments
- how mature communities treat standards as a layered portfolio rather than a
  single file format
